How to compile and run the code (Linux Environment):

Step 1 : 	In command line, execute "javac *.java" to compile all .java files (This step can be ignored. The lab4_100040X.zip file 		contains all compiled .class files)
Step 2 : 	Execute "java Main" to run the Main class which has the main method
Step 3 : 	Input the number of passengers to be boarded to the busses

The program will then execute and show how many passengers are boarded in each bus and will terminate when all passengers have boarded a bus.
